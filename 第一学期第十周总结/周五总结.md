#### ###位运算逐位读取二进制###
使用了**stdlib.h**库里面的**itoa**函数。</br>
把代码黏上来
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
	int i = 1;
	char x[100];
	for (; i <= 50;i++)
	{
		printf("%d的二进制为%s\n",i,x);
		itoa(i, x, 2);
	}
	return 0;
}
```
